# Rephoto AI Development Rules

This project uses Flutter.

Tech stack:
- Flutter
- Riverpod
- Isar Database

Rules:
- Do not modify unrelated files
- Do not refactor the entire project
- Keep folder structure stable
- Use Material 3
- Prefer mobile-first UI
- Prefer component-based widgets
- Avoid overly complex architecture
- Keep code beginner-friendly
- Add comments for important logic
