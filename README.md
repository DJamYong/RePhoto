# RePhoto — 每天翻一张旧照片，写下此刻的感受

手机里几千张照片，有多久没看过了？

RePhoto 每次随机翻出一张旧照片呈现在你面前。很久没被翻到的照片会优先出现，每张照片都有被重新看见的机会。

看到它，写几句话，选一个心情。记录一直跟着照片，下次翻到时还能看到当时的自己。

## 核心功能

- **每日回忆** — 随机抽选旧照片，权重偏向长期未翻阅的照片
- **心情日记** — 为照片添加文字、情绪标签和颜色标记
- **时间对撞** — 同月同日跨年照片并排对比，一眼看遍岁月变化
- **月度回顾** — 每月自动生成回忆报告，翻看次数、记录天数、情绪分布
- **全屏浏览** — 点击照片全屏查看，支持缩放和拖动
- **本地存储** — 所有数据在你的设备上，不联网，不上传，支持备份导入

## 隐私

RePhoto 不需要网络权限。你的照片和记录始终只属于你。

## 技术栈

Flutter · Riverpod · SQLite · photo_manager

## 构建

```bash
flutter pub get
flutter build apk --release
```

> APK 位于 `build/app/outputs/flutter-apk/app-release.apk`

|------|------|

| 相册写入 | 删除确认移除的照片 |
