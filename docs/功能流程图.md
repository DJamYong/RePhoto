## 1. 应用总览

```
main.dart
  │
  └─ ProviderScope (Riverpod 根)
       │
       └─ RePhotoApp (app.dart)
            ├─ 监听 themeModeProvider → 选择 light/dark 主题
            └─ 首页 HomePage
```

**关键依赖：**
- `flutter_riverpod ^3.3.1` — 状态管理
- `photo_manager ^3.3.0` — 设备相册访问
- `exif ^3.3.0` — EXIF 元数据解析
- `hive ^2.2.3` — 已声明但当前版本未使用

## 2. 启动流程

```
应用启动
  │
  ├─ Flutter 引擎初始化 (WidgetsFlutterBinding.ensureInitialized)
  │
  ├─ ProviderScope 注入
  │
  └─ RePhotoApp.build()
       │
       ├─ ref.watch(themeModeProvider)
       │     ├─ ThemeMode.system → 跟随系统
       │     ├─ ThemeMode.light → 暖色主题
       │     │     ├─ primary: #C08552
       │     │     ├─ surface: #FEF7EF
       │     │     └─ scaffold: #FDF6EC
       │     └─ ThemeMode.dark → 深色主题
       │           ├─ surface: #2C2420
       │           └─ scaffold: #1F1A17
       │
       └─ return HomePage()
             │
             └─ ref.watch(photoProvider)
                   │
                   ├─ 首次 build() → _loadRandomPhoto()
                   │     ├─ requestPermission() → PhotoManager
                   │     │     ├─ 已授权 → continue
                   │     │     └─ 未授权 → 返回 RandomPhotoState(hasPermission: false)
                   │     │
                   │     └─ getRandomPhoto()
                   │           ├─ getAssetPathList(type: image, hasAll: true)
                   │           ├─ 找到"所有照片"相册
                   │           ├─ assetCountAsync → 随机索引
                   │           ├─ getAssetListRange(start, end) → 1 张
                   │           └─ return AssetEntity? (null = 空相册)
                   │
                   └─ AsyncNotifier 状态
                         ├─ AsyncLoading    → _WarmLoading 骨架屏
                         ├─ AsyncData       → _PhotoAlbumView
                         └─ AsyncError      → _ErrorView
```

## 4. 照片加载流程

### 4.1 首次加载

```
PhotoProvider.build()
  │
  └─ _loadRandomPhoto()
       │
       ├─ PhotoService.requestPermission()
       │     └─ PhotoManager.requestPermissionExtend()
       │           └─ 返回权限状态
       │
       ├─ PhotoService.getRandomPhoto()
       │     ├─ getAssetPathList(type: image, hasAll: true)
       │     ├─ 找到 isAll 相册
       │     ├─ assetCountAsync → Random().nextInt(count)
       │     └─ getAssetListRange(start, end) → AssetEntity
       │
       └─ RandomPhotoState(photo, hasPermission)
```

### 4.2 预加载（点击换一张后触发）

```
PhotoProvider.refresh()
  │
  ├─ _loadRandomPhoto() → state = AsyncData(next)  ← 照片立即显示
  │
  └─ 后台预加载（不阻塞 UI）
        │
        ├─ Future.wait([
        │     photo.thumbnailDataWithSize(320×320, quality: 90),
        │     photo.file
        │   ])
        │
        ├─ readExifFromFile(file) → preloadedExif
        │
        └─ state = AsyncData(next.copyWith(
               preloadedThumbnail: thumb,
               preloadedFile: file,
               preloadedExif: exif
             ))
              │
              └─ 详情面板打开时直接使用预加载数据
```

## 5. 换一张流程

```
用户点击"换一张"
  │
  ├─ 旧：state = AsyncLoading → 闪烁转圈 ❌
  │
  └─ 新：直接异步替换（无转圈）
        │
        ├─ _loadRandomPhoto()  → 新 RandomPhotoState
        ├─ state = AsyncData(next)  → 照片立即切换
        │
        └─ 后台预加载详情数据
              ├─ thumbnailDataWithSize
              ├─ photo.file
              └─ readExifFromFile → preloadedExif
```

## 6. 删除照片流程

```
用户点击 🗑️ 删除按钮
  │
  ├─ showDialog 确认弹窗
  │     ├─ "取消" → 关闭弹窗，不执行任何操作
  │     └─ "删除" → 继续
  │
  ├─ PhotoService.deletePhoto(photo)
  │     └─ PhotoManager.editor.deleteWithIds([photo.id])
  │           ├─ Android 10+ → 系统弹窗二次确认
  │           └─ success → isNotEmpty
  │
  ├─ 删除成功后 → refresh() → 自动换到下一张
  │
  └─ 删除失败 → 无额外处理（返回 false）
```

## 10. 状态与异常处理

### 10.1 全局状态树

```
ProviderScope
  │
  ├─ photoProvider
  │     └─ AsyncNotifierProvider<PhotoProvider, RandomPhotoState>
  │           ├─ photo: AssetEntity?
  │           ├─ isLoading: bool
  │           ├─ errorMessage: String?
  │           ├─ hasPermission: bool
  │           ├─ preloadedThumbnail: Uint8List?
  │           ├─ preloadedFile: File?
  │           └─ preloadedExif: Map<String, IfdTag>?
  │
  ├─ themeModeProvider
  │     └─ NotifierProvider<ThemeModeNotifier, ThemeMode>
  │           └─ system / light / dark
  │
  └─ photoDisplayPrefsProvider
        └─ NotifierProvider<PhotoDisplayPrefsNotifier, PhotoDisplayPrefs>
              ├─ showDate: bool
              └─ showTitle: bool
```

### 10.2 所有 UI 状态总表

| 状态 | UI 组件 | 触发条件 |
|---|---|---|
| 应用加载中 | `_WarmLoading` | `photoAsync.when(loading)` — 首次启动 |
| 权限未授予 | `_ErrorView` | `state.hasPermission == false` |
| 相册为空 | `_ErrorView` | `state.photo == null && hasPermission` |
| 加载错误 | `_ErrorView` | `photoAsync.when(error)` |
| 照片就绪 | `_PhotoAlbumView` | `state.photo != null` |
| 缩略图加载中 | `CircularProgressIndicator` | `_PhotoWidget._isLoading` |
| 缩略图失败 | `broken_image` 图标 | `_PhotoWidget._hasError` |
| 详情面板加载中 | `CircularProgressIndicator` | 面板 `photoAsync.when(loading)` |
| 详情面板空 | "暂无照片" | `state.photo == null` |
| 详情面板错误 | "加载失败" | 面板 `photoAsync.when(error)` |
| 预加载进行中 | 回退异步加载 | `preloaded*` 数据为 null |
| 预加载完成 | 直接渲染 | `preloaded*` 数据就绪 |
