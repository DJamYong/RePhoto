随机相册回忆APP（Flutter）需求文档
1. 项目概述
1.1 项目名称
RePhoto | 重逢

1.2 项目目标

开发一款跨平台（Android + iOS）的 Flutter 应用，从用户本地相册中每日随机抽取若干照片进行展示，并允许用户对照片进行回忆记录管理。

核心目标：

提升用户“照片回忆触发体验”
将“随机浏览照片”变为“带时间线的记忆回顾”
2. 技术栈要求
前端框架：Flutter（必须）
语言：Dart
平台：Android / iOS（单代码库）
本地存储：
Hive
相册访问：
photo_manager 或 image_picker
权限管理：
相册权限（Android + iOS）
3. 核心功能模块
从用户相册中随机抽取 N 张照片（默认 3～6 张）
